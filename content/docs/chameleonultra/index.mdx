# 读卡

## 判断卡片类型

使用变色龙读取卡片，高频能读到就是 IC 卡，低频能读到就是 ID 卡。注意读高频使用正面（滴胶面）贴近卡片，读低频使用背面（图案面）贴近卡片，两面板子对应不同的频率。

其中 IC 卡根据 SAK 响应码可分为：

- `08`代表 S50 卡
- `18`代表 S70 卡
- `20`代表 CPU 卡
- `28`代表 CPU 模拟卡

变色龙客户端层面目前仅支持 S50 卡，其他类型卡片请参考项目使用命令行等方式测试，需要有一定的代码基础。

## IC 卡读取

IC 卡根据加密方式可分为：

- 非加密 IC 卡
- 半加密 IC 卡
- 全加密 IC 卡

我们采取依次尝试的办法可以确定卡片加密方式，并破解卡片

## ID 卡读取

ID 卡因为没有数据区域也没有密码，所以读取非常简单，只需要读取卡号保存即可。

# 写数据到变色龙

1. 选择卡槽管理页面
2. 点击一个未使用的卡槽，弹出已保存的卡片列表
3. 点击卡片即可写入
